La présentation du projet

Ce projet a été réalisé dans le cadre de l’UV NF28 - Ingénierie des systèmes interactifs durant le semestre de printemps 2017.

Le projet a été proposé par des doctorants de l’UTC travaillant sur des serious game.

Ces jeux nécessitent l’utilisation de fichiers XML contenant les tâches à réaliser et les conditions nécessaires à leur réalisation. Les arborescence de tâches doivent pouvoir être réalisée par des ergonomes sans bagages informatique. Ils ne sont donc pas capables de réaliser le fichier XML. L’idée est donc de réaliser un logiciel leur permettant de réaliser cette arborescen

Un logiciel de ce type avait déjà été réalisé, mais il n’était pas simple d’utilisation.

Nous avons donc recréé une interface afin de répondre à ce besoin. Notre groupe se compose de Théo Judas, Pierre-Louis Lacorte, Camille Soetaert et Oumaima Talouka.
